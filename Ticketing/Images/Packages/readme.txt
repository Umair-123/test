-------------------------
setting up a combo image
-------------------------

* look at example.gif for image width/height
* you can have an image with different width/height but you'll need to change
  the dimensions in "visStyles.css", style "ImageVoucher"
* make the graphic a ".gif" or ".jpg"
* name the graphic the same as the combo
